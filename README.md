# at-hth-archive

## 简介

at-hth-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，仅作为 HTML 资源的集中存储与展示空间。

这些页面可能来自不同来源，或为个人练习、项目备份、静态内容存档等用途。所有内容以原始 HTML 格式保留，便于直接查看或部署。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源文件（如有）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个文件为一个独立页面。
- `assets/` 目录用于存放页面可能引用的公共样式、脚本或图片资源。
- 索引页面（`index.html`）提供对归档页面的导航或列表。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部动态服务。
- 页面内容可能包含静态文本、样式、脚本及本地资源引用。
- 归档页面可能来自不同时期或项目，格式与风格可能不统一。
- 页面内容不涉及具体网站、域名或商业推广，仅为技术性存档。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有内容。
- 如需提交新页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的索引（如有）。
- 欢迎通过 Issue 或 Pull Request 提出建议或报告问题。
- 本仓库为个人归档用途，不保证长期可用性或内容完整性。
